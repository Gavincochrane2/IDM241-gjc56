IDM 241 Final Description

Link to microinteraction: Whatsapp Download panels

Desktop


Mac



Requires macOS 11 or newer. For all other operating systems,

you can use WhatsApp Web in your browser










  • Triggers:
  • Hover on download button
  • Hover off download button
  • Hover on download panel
  • Hover off download panel
  • Click download button
  • Rules:
  • Hover on button
  • The button changes color by sliding from the bottom from green to white
  • The cursor changes to a pointer
  • Hover on panel
  • The panel changes color abruptly from black to green
  • The text changes color from white to black
  • Hover off button
  • The button color changes radially from the bottom from white to green
  • The pointer changes back to the normal cursor
  • Hover off panel
  • The panel abruptly changes color from green to black
  • The text changes color from black to white
  • Click download button
  • Button shrinks from either side and transforms into a loading icon that spins
  • Feedback:
  • Hover on button
  • The button changes color radially from the bottom from green to white
  • The cursor changes to a pointer
  • Hover on panel
  • The panel changes color abruptly from black to green
  • The text changes color from white to black
  • Hover off button
  • The button color changes radially from the bottom from white to green
  • The pointer changes back to the normal cursor
  • Hover off panel
  • The panel abruptly changes color from green to black
  • The text changes color from black to white
  • Click download button
  • Button shrinks from either side and transforms into a loading icon that spins
  • Loops: loading animation loops infinitely until the user clicks off

  • Modes: new loading icon shows up on click
